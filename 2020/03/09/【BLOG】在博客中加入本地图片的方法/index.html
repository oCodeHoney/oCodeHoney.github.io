刚刚写一篇博文时尝试将本地的图片以平时写HTML和CSS时同样的相对路径法插入，结果无法显示，原来还需要一些操作。

参考

  • hexo引用本地图片无法显示

方法

首先安装插件

npm install https://github.com/CodeFalling/hexo-asset-image --save

打开/node_modules/hexo-asset-image/index.js，将内容更换为下面的代码

打开_config.yml文件，修改下述内容

post_asset_folder: true

在/sourse/_posts文件夹中创建一个与xxxx.md文件同名的文件夹xxxx，将所需的图片xxx.jpg放入文件夹中，在md中以xxxx/xxx.jpg进行引用。
